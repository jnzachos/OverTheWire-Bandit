# OverTheWire-Bandit by jnzachos
OverTheWire Bandit walkthrough writeups to help anyone learn terminal basics and Linux fundamentals.

---

### Instructions

Go to https://overthewire.org/wargames/bandit/ and read the `Note for beginners`, then try solving the challenges.

If you find yourself stuck, or do not know what to do, go to the `writeups` directory, choose the level you are at and read the corresponding writeup for a hint or solution.

`Note: The writeups are fairly detailed, so simply following them might ruin the fun. Try solving with your own skills first.`
